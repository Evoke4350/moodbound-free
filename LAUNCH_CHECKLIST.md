# Launch Checklist

Status legend: `[x] done` `[ ] pending`

## Product Readiness

- [x] Core check-in create/edit/delete flows working.
- [x] Insights and safety warning surfaces present.
- [x] Safety plan and support contact flows available.
- [x] Backup export/import path available.
- [ ] Reminder UX validated on physical devices.

## Quality Gates

- [x] Unit test suite passing.
- [x] Integration test suite passing.
- [x] UI smoke tests passing.
- [x] CI workflow with coverage floor present.
- [ ] Manual exploratory QA pass completed.

## Reliability and Safety

- [x] Non-fatal diagnostics logging in place.
- [x] Validation guardrails enforced on writes.
- [x] Safety copy guardrail policy implemented.
- [ ] Clinical language review by domain expert completed.

## Release Operations

- [x] Performance budget document present.
- [x] Observability runbook present.
- [x] Sync/conflict strategy documented.
- [ ] Release rollback owner assigned.
